- **Actualización:** Se actualiza constantemente con nuevas incorporaciones

## **Sistema de Filtros**

### **Filtros Principales**

**Por Categorías**
- Selecciona una o múltiples categorías principales
- Refina por subcategorías específicas
- Combina categorías relacionadas para búsquedas amplias

**Por Marcas**
- Filtra por marcas específicas de tu preferencia
- Muestra todas las marcas disponibles en cada categoría
- Facilita encontrar productos de fabricantes confiables

**Por Tiendas en Douvery**
- Filtra productos por tiendas específicas
- Encuentra vendedores de tu confianza
- Explora el catálogo completo de tiendas particulares

**Por Condición del Producto**
- **Nuevo:** Productos sin usar, en empaque original
- **Usado:** Productos previamente utilizados en diferentes estados

### **Opciones de Ordenamiento**

Organiza los resultados según tus preferencias:

<AccordionGroup>
  <Accordion title="Por Relevancia">

### **Configuración de Visualización**

**Cantidad de Productos por Página**
- Personaliza cuántos productos ver por página
- Opciones típicas: 12, 24, 48, 96 productos
- Más productos = menos clics, pero carga más lenta
- Menos productos = navegación más rápida

### **Combinación de Filtros**

**Filtros Múltiples**
Puedes combinar diferentes filtros para búsquedas muy específicas:

**Ejemplo de Combinación:**
- **Categoría:** Electrónicos → Teléfonos
- **Marca:** Samsung
- **Tienda:** Tienda específica de confianza
- **Condición:** Solo productos nuevos
- **Ordenar por:** Precio menor a mayor

**Consejos para Filtros Efectivos:**
- Comienza con filtros amplios y ve refinando
- Si no hay resultados, elimina algunos filtros
- Usa el ordenamiento para priorizar lo que más te importa
- Guarda combinaciones de filtros que uses frecuentemente

## **Navegación Avanzada**

### **Búsqueda dentro de Categorías**
- Usa la barra de búsqueda mientras navegas una categoría específica
- Los resultados se limitarán a esa categoría
- Ideal para búsquedas muy específicas dentro de un área

### **Breadcrumbs (Migas de Pan)**
- Rastrea tu ubicación actual en el catálogo
- Navega fácilmente hacia atrás en la jerarquía
- Ejemplo: Inicio → Electrónicos → Teléfonos → Samsung

### **Filtros Guardados**
- Guarda combinaciones de filtros frecuentes
- Acceso rápido a búsquedas repetitivas
- Personaliza tu experiencia de navegación

## **Consejos de Navegación**

### **Para Compras Generales**
1. **Comienza por categorías amplias**
2. **Usa las secciones especiales** (ofertas, populares, nuevos)
3. **Refina con filtros** según tus necesidades
4. **Experimenta con el ordenamiento** para diferentes perspectivas

### **Para Búsquedas Específicas**
1. **Ve directamente a la subcategoría** relevante
2. **Usa filtros de marca** si tienes preferencias
3. **Combina múltiples filtros** para precisión
4. **Ajusta la cantidad por página** según tu paciencia

### **Para Descubrir Productos**
1. **Explora "Más Populares"** para ver tendencias
2. **Visita "Productos Nuevos"** regularmente
3. **Navega sin filtros** ocasionalmente para sorpresas
4. **Prueba categorías** que normalmente no visitas

<Note>
  **Consejo Pro:** Los filtros se pueden combinar de múltiples maneras. Experimenta con diferentes combinaciones para encontrar exactamente lo que buscas.
</Note>
